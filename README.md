## Tech Stack

**Client:** 
- HTML
- CSS 
- Javascript
- ReactJS

**Backend** :
  - [Mockbee](https://mockbee.netlify.app/)


**Hosted** :
- Netlify

## Library Used
  - [NOMARK UI](https://nomark-ui.netlify.app/)


  
## Contains

  - Home Page 
  - Product Page
  - Wishlist Page
  - Cart Page
  - Login Page
  - Signup Page

## Features
  - Category Listing
  - Product Listing
  - Filter Products by
    - Category
    - Price
    - Rating
  - Sort Products by
    - Price: Low to High
    - Price: High to Low  
  - Cart Management
  - Wishlist Management
